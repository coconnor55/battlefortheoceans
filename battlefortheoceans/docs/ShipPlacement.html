Ship Placement Board CSS Updates

  • Medium green (#90EE90) for valid placement preview

  • Light blue (#ADD8E6) for cells with placed ships
  • No background box behind the grid
  • NOAA Chart 1 terrain colors for different water types
  • Responsive design for mobile devices
